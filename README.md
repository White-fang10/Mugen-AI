| `cost_estimate` | "2 grand" | `2000.0` | вҖ” | 0.91 | none |

| Confidence | Action |
|:---:|:---|
| `вүҘ 0.70` | вң… Slot accepted |
| `0.40 вҖ“ 0.69` | рҹ”Қ Re-ask with contextual hint |
| `< 0.40` | вқ“ Re-ask with original prompt (max 3 retries) |
| `injection_risk: high` | рҹ”’ Session permanently frozen |


```bash
  -v $(pwd)/data:/app/data \
  -v $(pwd)/rulebooks:/app/rulebooks \
  -v $(pwd)/chroma_store:/app/chroma_store \
```
